# TASK-3-Web-Scraper-for-News-Headlines-Elevate-Labs-Internship-
A simple Python project created for Task 3 of the Elevate Labs Internship. This script scrapes the latest news headlines from a public website using requests and BeautifulSoup, then saves the extracted headlines into a headlines.txt file. It demonstrates basic web scraping, HTML parsing, HTTP requests, and file handling in Python.
